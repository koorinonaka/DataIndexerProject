# DataIndexerProject

Sample project and documentation for the [DataIndexer](https://github.com/koorinonaka/DataIndexer) Unreal Engine plugin.

## Setup

1. Clone with submodules:

```bash
git clone --recurse-submodules https://github.com/koorinonaka/DataIndexerProject.git
```

2. Regenerate project files.
3. Build the project.
4. Open `DataIndexerProject.uproject` in Unreal Engine 5.4+.

## Documentation

See the `docs/` directory. To serve locally:

```bash
cd docs
pip install -r requirements.txt
mkdocs serve
```
